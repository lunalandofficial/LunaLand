# LunaLand

LunaLand (LLN) is a decentralized hyper deflationary peer-to-peer digital currency.

A vibrant community-driven token that rewards long-term holders through game-changing reward features.

LunaLand, a store of value providing real-world inflation protection and promoting no-tax, low slippage trading.

There are a number main components of LunaLand that make LunaLand stand out from other hypedeflationary tokens.

1. Supply is currently only 320 Million with ultimate supply 50 Million
2. Unique community rewards including:- Burn and Earn, Luna Calendar Internal Airdrops, Exclusive LunaLand NFT's and Rise and Prize. 
3. Focus on real world utility by building an e-commerce store (LunaLand Store) that will accept LunaLand tokens as currency payments.
